# todo-list

one-pager

This is a todo app which allows you to:

create tasks with: 
* descriptions
* due date
* priority

You will be able to create separate project 'pages' in which to create 
relevant lists of tasks. there will be a task overview page which shows ALL tasks
in one location.


input -> computation -> output

user inputs:

* new project
* new task -->
    > due date
    > priority
    > description
    > close task

computation:

* todo task factory
    > with props:
        > task title
        > task description
        > task priority
    > with methods:
        > edit title
        > edit description
        > edit priority
        > close task
